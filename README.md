# Portfolio

Este é o meu portfólio com formulário de contato funcional, enviando emails via **Resend API**.  

---

## Atualização do Frontend

Sempre que atualizar o frontend, execute:

```
npm run build
npm run deploy
```

## Tech Stack

Frontend: React, JavaScript, React Bootstrap, animate.css, react-on-screen

Backend: Node.js, Express.js, CORS, dotenv

Email: Resend API (envio de emails via HTTP)

Deploy / Infra: Render (auto deploy via GitHub)
